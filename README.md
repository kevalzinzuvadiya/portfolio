## 🛠️ Tech Stack

### Frontend
| Technology | Purpose |
|------------|---------|
| **HTML5** | Semantic page structure and SEO-optimized markup |
| **CSS3** | Custom design system with CSS variables, glassmorphism, gradients, and responsive layouts |
| **JavaScript (Vanilla)** | Interactive animations, particle system, scroll effects, and DOM manipulation |

### Design System
| Feature | Implementation |
|---------|---------------|
| **Typography** | [Inter](https://fonts.google.com/specimen/Inter) (UI) + [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) (code/accents) via Google Fonts |
| **Color Palette** | Dark theme with teal (`#00d4aa`) and purple (`#6c63ff`) accent gradients |
| **Glassmorphism** | `backdrop-filter: blur()` with semi-transparent backgrounds |
| **Animations** | CSS transitions + Intersection Observer API for scroll-reveal effects |
| **Particles** | Custom HTML5 Canvas particle system with connecting lines |

### Key Features
- ✨ **Animated Particle Background** — Canvas-based particle system with dynamic connections
- 🎨 **Dark Mode Design** — Premium dark UI with teal/purple gradient accents
- 📱 **Fully Responsive** — Optimized for mobile (375px), tablet (768px), and desktop (1024px+)
- 🔮 **Glassmorphism Cards** — Modern frosted-glass card design with hover effects
- ⚡ **Scroll Reveal Animations** — Elements animate into view using Intersection Observer
- 🎯 **Smooth Navigation** — Sticky navbar with smooth scroll and hamburger menu for mobile
- 💬 **Contact Form** — Interactive form with send confirmation animation
- ♿ **Accessible** — Respects `prefers-reduced-motion`, semantic HTML, focus states

## 🚀 Getting Started

Clone the repository and open in any browser:

```bash
git clone https://github.com/kevalzinzuvadiya/portfolio.git
cd portfolio
```

**Option 1:** Open `index.html` directly in your browser.

**Option 2:** Serve locally with Python:
```bash
python -m http.server 3000
# Visit http://localhost:3000
```
